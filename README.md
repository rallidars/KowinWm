## About

**KowinWM** is a simple wayland  window manager created with rust lang and smithay framework

## Features

- [x] Nvidia Support
- [x] Dynamic Tiling
- [x] Keyboard and Monitor Managament
- [X] Borders
- [x] Can display wayland applications
- [ ] Xwayland Support
- [x] Working Popups


## Compilation
Clone the git repo and build KowinWM by running the following command:
```bash
$ cargo build --release
```
The binary will be created in `./target/release/kowinwm`.
